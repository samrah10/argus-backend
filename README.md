# Argus

Argus is a camera-based web app prototype for blind-user walking guidance.

## Features
- Names major obstacles in the path
- Calls out floor hazards like cables, mats, carpet edges, and other trip hazards
- Recommends a walking direction
- Gives a rough walking angle and short move distance
- Default auto-scan interval is 1 second

## Run
1. `npm install`
2. Create `.env` from `.env.example`
3. `npm start`
4. Open `http://localhost:3000`
